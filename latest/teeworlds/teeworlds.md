## Controls

| Button | Action |
|--|--|
| Dpad | Move |
| Left Analog | Move|
| Right Analog | Move |
| A | Dig Left |
| B | Dig Right |
| X | Space/Start |
| Y | Pick Level |
| L1 | Go to Previous Level  |
| L2 | Go to Next Level |
| R1 | Goto |
| R2 | Full Screen|
